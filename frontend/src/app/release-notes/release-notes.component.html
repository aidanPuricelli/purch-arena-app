1.2.0

This minor release includes:

  • Online Multiplayer!
    • Still in early development
    • Limited to two person games
    • A little confusing to set up (a guide will be created soon)

I'm releasing online multiplayer in it's unfinished state. I plan to refine/complete it and when done, I will release version 2.0.0.

Multiplayer functionality depends on ngrok, an auth token is necessary. You need to create a .env file in the root dir of the app, and enter your auth token with the following syntax: NGROK_AUTH_TOKEN=<your-auth-token>

You can still use the app in the same way you did before (screen sharing).
