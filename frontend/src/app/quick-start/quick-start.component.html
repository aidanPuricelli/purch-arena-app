PurchArena is an all-in-one platform for building and playing Magic: The Gathering decks, combining official cards with user-created custom cards. Welcome to version 1.2.0 :)

Building Your Deck

Getting Started

To start building, click the Build button to be navigated to the build page.

Next follow these steps to create a new deck:

  • Enter a name for your deck in the text area provided.
  • Click Create.

Once you've created your deck, you can search for cards using the search bar at the top of the page. Right click a card and select Add to Deck to add the card into your current deck.

Importing/Exporting Decks

The build page includes a feature to either import or export decks. To export a deck follow these steps:

  • Select a deck using the Select Deck dropdown.
  • Click the settings icon and select Download.
  • A JSON file named after your deck will be downloaded in your browser.

To import a deck, follow these steps:

  • Select a deck using the Select Deck dropdown, or create a new deck.
  • Click the settings icon and select either:
    • Import from JSON (if you have a file with type .json).
    • Import from text (if you have a .txt file, which is standard for other deck building clients like MoxField or ArchiDekt).
  • After selecting an option you will be prompted to upload a deck list.
  • Selecting and uploading a deck list will populate the current deck with those cards.

Deck lists from other deck building clients are supported.

Double-sided cards are currently not supported.

Other important features:

  • To remove a card from your deck, right click it and select Remove Card.
  • You can sort your deck by creature type or mana cost using the Sort By dropdown.
  • To designate a commander for your deck, right click a card and click Set Commander (this will move the card to the top of the deck).

Clicking Delete will erase the current deck, this action cannot be undone.
